# IA-Labyrinthe
 This Artificial Intelligence allows you to go through a maze in search of mussels. But it also seeks to recover the bonuses. 🦪🍟🍺

# To start a game 🎮

To begin, start the server. 

$: cd serveur_pecheMoules

$: java ServeurEpreuve1

Start a second terminal 

$: cd clientACompleter

$: javac Client.java

$: java Client 127.0.0.1 1337 <Your name, nickname, nickname>
